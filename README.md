# UK AI Investment Dashboard

A responsive, animated one-page dashboard showcasing UK AI investments. Built with React and featuring three distinct scroll-based sections.

## Features

- **Hero Section**: Animated "UK AI" title with dynamic kerning and font scaling on scroll
- **Statistics Section**: Animated statistics cards with hover effects
- **Investment Graph**: Interactive obsidian-style force graph showing UK AI investments with hover and click interactions

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

### Build

```bash
npm run build
```

## Technology Stack

- React 18
- Vite
- Framer Motion (animations)
- React Force Graph 2D (graph visualisation)
- PapaParse (CSV parsing)

## Design

- Base colour: #1B1B3A
- Responsive design with mobile support
- Smooth scroll-based animations
- Interactive graph visualisation
