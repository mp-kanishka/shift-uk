# Shift UK

An interactive dashboard that tracks and showcases the UK AI story.

## Features

- A graph visualisation of UK governemnt announcements on AI
- Animated UI components with Framer Motion
- CSV-based data management

## Technologies

- **React** - UI framework
- **Vite** - Build tool and dev server
- **react-force-graph-2d** - Network graph visualization
- **D3 Force** - Force simulation physics
- **Framer Motion** - Animation library
- **PapaParse** - CSV parsing

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Start the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:....`

### Build

Create a production build:
```bash
npm run build
```

### Preview

Preview the production build locally:
```bash
npm run preview
```
